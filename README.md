## 📖 Про проєкт

**Number Converter Pro** — це десктопний додаток на мові Java, призначений для швидкого та точного переведення чисел між різними системами числення (від 2 до 36). Проєкт поєднує в собі потужну математичну логіку для роботи з надвеликими числами та сучасний графічний інтерфейс.

-----

## ✨ Функціональні можливості

  * **Широкий діапазон основ:** Підтримка систем числення від двійкової (2) до тридцятишісткової (36).
  * **Робота з великими числами:** Завдяки використанню `BigInteger`, програма не має стандартних обмежень на довжину числа.
  * **Динамічні теми:** Можливість миттєвого перемикання між **Dark Theme** (темною) та **Light Theme** (світлою) без перезапуску програми.
  * **Сучасний UI:** Інтерфейс побудований на бібліотеці FlatLaf із закругленими елементами керування та ергономічним дизайном.
  * **Валідація:** Миттєве виявлення помилок введення та кольорова індикація результату.

-----

## 🛠 Використані технології

  * **Мова програмування:** Java (JDK 17+)
  * **Графічна бібліотека:** Java Swing
  * **Look and Feel:** [FlatLaf](https://www.formdev.com/flatlaf/) (FlatDarkLaf / FlatLightLaf)
  * **Математична логіка:** `java.math.BigInteger`
  * **Шрифти:** JetBrains Mono (для кращої читабельності чисел)

## 🚀 Інструкція зі встановлення та запуску

Проєкт розроблено в середовищі **Eclipse IDE**. Для запуску виконайте наступні кроки:

1.  **Клонуйте репозиторій** або завантажте вихідний код.
2.  **Імпортуйте проєкт в Eclipse:**
      - `File -> Import -> General -> Existing Projects into Workspace`.
3.  **Додайте бібліотеку FlatLaf:**
      - Переконайтеся, що файл `flatlaf-x.x.jar` доданий до **Build Path** проєкту.
4.  **Запуск:**
      - Знайдіть файл `ConverterApp.java` у пакеті `src`.
      - Натисніть правою кнопкою миші -\> `Run As -> Java Application`.

## 💡 Інструкція з використання

Слідуйте цим крокам для виконання конвертації:

1.  **Оберіть візуальний режим:** Натисніть кнопку у верхній частині вікна (**Light Theme / Dark Theme**) для комфортної роботи.
2.  **Введіть число:** У текстове поле "Вхідні дані" впишіть число, яке необхідно перетворити.
3.  **Оберіть системи числення:**
      - У лівому списку (**З системи**) оберіть основу вашого числа.
      - У правому списку (**В систему**) оберіть цільову основу.
4.  **Отримайте результат:** Натисніть кнопку **КОНВЕРТУВАТИ**.
      - **Зелений колір:** Успішна конвертація. Результат з'явиться у нижньому полі.
      - **Червоний колір:** Помилка (некоректні символи для обраної системи).

## ⚠️ Обробка помилок та валідація

Програма має вбудований захист від некоректного введення:

  - Якщо ви спробуєте перевести число `102` у двійковій системі (де дозволені лише 0 та 1), додаток не "висне", а виведе повідомлення **"Помилка даних\!"**.
  - Весь процес обробки захищений блоком `try-catch (NumberFormatException)`, що забезпечує стабільність роботи програми навіть при випадкових помилках користувача.
